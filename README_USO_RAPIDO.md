# README_USO_RAPIDO.md

## Para qué sirve este paquete

Este paquete existe para evitar perder contexto cuando una conversación larga se cae, se borra o toca empezar un chat nuevo.

## Orden recomendado de uso

### Opción completa
Comparte en un chat nuevo:
1. `BOOTSTRAP_PROMPT.md`
2. `PROJECT_CONTEXT.md`
3. `SESSION_HANDOFF.md`
4. `KNOWN_ISSUES.md`
5. `TEST_STATUS.md`

### Opción rápida
Si tienes poco tiempo, comparte:
- `BOOTSTRAP_PROMPT.md`
- `PROJECT_CONTEXT.md`
- `SESSION_HANDOFF.md`

## Cuándo actualizar cada archivo

### `PROJECT_CONTEXT.md`
Actualízalo solo cuando cambie algo estructural:
- nombre del proyecto
- alcance
- stack
- módulos
- reglas base

### `SESSION_HANDOFF.md`
Actualízalo cada vez que cierres una sesión importante.

### `KNOWN_ISSUES.md`
Actualízalo cuando confirmes nuevos bugs o soluciones problemas existentes.

### `TEST_STATUS.md`
Actualízalo cuando avances en:
- pruebas
- capturas
- evidencias
- veredictos

## Consejo práctico

Guarda estos archivos:
- dentro del repo
- y también en una carpeta externa de respaldo

Así, aunque pierdas un chat, no pierdes el cerebro del proyecto.
